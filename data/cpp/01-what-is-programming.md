# What is Programming?

## Introduction

Programming is the process of writing instructions that tell a computer what to do.

A computer cannot think like a human.

It cannot guess.

It cannot fill in missing information.

It only follows instructions exactly as they are written.

---

## Computers Are Very Literal

Imagine telling someone:

> Make me a sandwich.

A person knows what you probably mean.

A computer does not.

You would need to explain every step:

1. Pick up bread.
2. Open peanut butter.
3. Spread peanut butter.
4. Put bread together.

Programming works exactly the same way.

A program is simply a list of instructions.

---

## What Can Programming Create?

Programming is used to create:

- Websites
- Games
- Mobile apps
- Robots
- Artificial Intelligence
- Operating Systems

Almost everything electronic contains software.

---

## What Is C++?

C++ is a programming language.

Programming languages allow humans to communicate with computers.

C++ is particularly popular because it is:

- Extremely fast
- Used professionally
- Great for making games
- Excellent for learning how computers work internally
